Project Summary
This project compares classical machine learning models (Logistic Regression and SVM using TF-IDF features) with a transformer-based model (DistilBERT) for sentiment classification on the IMDB dataset.

We evaluate:
• Predictive performance
• Probability calibration
• Model interpretability using saliency maps

Dataset
IMDB Large Movie Review Dataset.

Models
Logistic Regression (TF-IDF)
SVM (TF-IDF)
DistilBERT
Temperature Scaling
Isotonic Regression

Notebook
project.ipynb demonstrates:
• Model evaluation
• Calibration analysis
• Saliency map examples

Repository Structure
